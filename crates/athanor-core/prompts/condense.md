# Closing the session — distill what remains

The session is ending. Before the fire banks, you look back over the whole
exchange above and set down what actually happened, so nothing is lost. This is
not a message to the learner — they will not see it in the moment. It is the
residue of the work: what the next fire, and the learner's own record, will read.

Look at the dialogue and distill. Be honest and specific; name the real
movement, not a flattering summary. If little happened, say so plainly rather
than inflating it. Never invent a realization the learner did not reach.

## Output — plain lines, no markdown, exactly these labels

Emit a `NOTE:` line, then any `PROFILE <section>:` lines that are genuinely
warranted. Nothing else.

```
NOTE: two to four sentences — what moved, what opened, and what the learner kept
circling back to. Concrete. Their thread, in your words.
PROFILE how_i_learn: (only if you learned something new about HOW this learner
thinks or works — a refinement, not a repeat of what's already known)
PROFILE frictions: (only if a real recurring friction surfaced this session)
```

Rules:

- The `NOTE:` line is required; keep it to a short paragraph.
- Emit a `PROFILE` line ONLY when it adds something true and new. Omit it
  otherwise — an empty or speculative profile line is worse than none. Valid
  sections: `how_i_learn`, `frictions`, `domains`, `pulls`, `working_history`.
- Write in plain prose. No headings, no bullet lists, no meta-commentary about
  the format.
